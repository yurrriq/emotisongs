** Installation
Just add it to your =rebar.config= deps:
#+BEGIN_SRC erlang
{deps, [
   ...
   {emotisongs, {git, "git://github.com/yurrriq/emotisongs.git", {branch, "develop"}}}
  ]}.
#+END_SRC

And then do the usual:
#+BEGIN_SRC bash
    $ rebar get-deps
    $ rebar compile
#+END_SRC
